# AI Dev Tools

IDE-level features and configuration surfaces for AI-assisted development — focused on Kiro (AWS's AI IDE) and comparable patterns in other tools.

## Articles

- [[kiro-hooks|Kiro Hooks]] — Event-driven automation that triggers agent actions on file saves, tool calls, and other IDE events
- [[kiro-specs|Kiro Specs]] — Structured specifications that decompose features/bugs into requirements → design → tasks
- [[kiro-steering|Kiro Steering]] — Persistent markdown files that give the AI context about your workspace, stack, and conventions
